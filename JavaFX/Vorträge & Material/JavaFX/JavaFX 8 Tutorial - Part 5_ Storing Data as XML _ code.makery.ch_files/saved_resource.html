                        Marco Jakob Mod Felipe Lopez 4 years ago

                        (It's always helpful if you include the error message in your question.)

                        I guess the problem might be that you are mixing static and non-static things. You static method "getPersonData", because it is static, only exists once for this class. Now you can't access an instance variable like "this.personData". An instance variable might exist multiple times for this class (one in every new object you create). So the compiler wouldn't know which one to pick for you...

                        A general advice: Even if you get it to work, making a method like "getPersonData" static is not such a good idea. Something that is "public static" can get accessed from anywhere in your entire project. When the project grows you'll soon have many different classes wildly accessing public static things everywhere. This quickly leads to "spaghetti code".
