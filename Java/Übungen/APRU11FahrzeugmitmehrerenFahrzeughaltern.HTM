Übung 10

Aufgabe Arrays

Ausgangspunkt dieser Aufgabe ist die Lösung der Übung 7

Ein Fahrzeughalter kann bisher nur ein Fahrzeug halten. Ändern Sie die Klasse so ab, dass ein Fahrzeug beliebig viele Fahrzeuge halten kann. Anstelle der 1 : 1 Beziehung zwischen der Klasse Fahrzeug und Fahrzeughlater tritt jetzt eine 1 : * Beziehung.

Verwenden sie ein Array, um die Fahrzeuge zu speichern. Das Array enthält keine Leerestellen. Immer wenn ein Fahrzeug hinzukommt, wird ein neues Array angelegt, dam um eine Komponente größer ist als das alte, und der Inhalt des alten Arrays wird in das neue Array übernommen.

Das Hauptprogramm sieht dann z.B. so aus:

Fahrzeughalter berty = new Fahrzeughalter("Berty","Vogts");
Fahrzeughalter uwe = new Fahrzeughalter("Uwe","Seeler");

new PKW(berty,4,5);
new Motorrad(berty);
new PKW(uwe,4,7);
new Motorrad(berty);
new Motorrad(berty);


System.out.println("Die Fahrzeuge von "+berty+":");
Fahrzeug[] fzgeBerty = berty.getFahrzeug();
for(int i=0;i<fzgeBerty.length;i++)
{
System.out.println(fzgeBerty[i]);
}
System.out.println("Die Fahrzeuge von "+uwe+":");
Fahrzeug[] fzgeUwe = uwe.getFahrzeug();
for(int i=0;i<fzgeUwe.length;i++)
{
System.out.println(fzgeUwe[i]);
}

mit dieser Ausgabe:

Die Fahrzeuge von Berty Vogts:

Das Fahrzeug 1 gehört Berty Vogts. Es ist ein PKW mit 4 Rädern und 5 Türen.

Das Fahrzeug 4 gehört Berty Vogts. Es ist ein Motorrad mit 2 Rädern.

Das Fahrzeug 5 gehört Berty Vogts. Es ist ein Motorrad mit 2 Rädern.

Die Fahrzeuge von Uwe Seelers:

Das Fahrzeug 2 gehört Uwe Seeler. Es ist ein Motorrad mit 2 Rädern.

Das Fahrzeug 3 gehört Uwe Seeler. Es ist ein PKW mit 4 Rädern und 7 Türen.
