Übung 15

Aufgabe Speichern in Dateien

Ausgangspunkt dieser Übung ist die Lösung der Übung 12.

Zur Zeit gehen alle unsere Daten in den Klassen Ware, Kunde, Adresse, Lieferant und Kundenrabatt, Bestellung und Bestellposition beim Beenden des Programms verloren (d.h. die Daten sind transient).

Speichern  sie die Daten in Dateien (d.h.die Daten sind persistent), verwenden sie für jede Klasse eine eigene Datei.

Sobald eine Änderung im Datenbestand gemacht wird, soll der aktuelle Stand wieder gesichert werden.

Beim Starten des Programms soll der Datenbestand aus den Dateien befüllt werden.

Kapseln sie dabei den Zugriff auf die Dateien mit Methoden so, dass es in Zukunft egal ist wo die Daten gespeichert bzw. ausgelesen werden.
