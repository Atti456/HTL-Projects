Entwicklen sie eine Klasse Kurzzeitwecker.

Einen Kurzeitwecker kennen Sie aus der Küche. Er wird auf eine bestimmte Zeit gestellt und läuft dann ab. Wenn er abgelaufen ist, ertönt ein Klingel- oder anderes Zeichen.

Die Klasse hat folgende Attribute:

private int min = 0;

private int sek = 0;

private int zehntelsek = 0;

und folgende Methoden:

public void stelle(String time)

Stellt die Zeit im Kurzeitwecker auf die spezifizierten Werte.

public void ticke()

Lässt den Wecker ticken, von der eingestellten Zeit bis 0:0,0. Wenn die Zeit abgelaufen ist, wird eine Nachricht auf die Konsole ausgegeben.

public String gibAlsString()

Gibt den augenblicklichen Stand des Kurzzeitweckers in Form eines String Objkets zurück.

Schreiben Sie eine Testmethode main(), die so aussehen könnte:

public static void main(String[] args)

{

    Kurzzeitwecker w = new Kurzzeitwecker();

    w.stelle(01:12,4);

    w.ticke();

}

Die Ausgabe sieht so aus:

Anfangszeit:            01:12,4

Verbleibende Zeit:    01:10,0

Verbleibende Zeit:    01:00,0

Verbleibende Zeit:    00:50,0

Verbleibende Zeit:    00:40,0

Verbleibende Zeit:    00:30,0

Verbleibende Zeit:    00:20,0

Verbleibende Zeit:    00:10,0

Die Zeit (01:12,4) ist abgelaufen.

Das waren 724 Zehntelsekunden.

Hinweise zur Implementierung:

In der Methode ticke() wird zu Beginn die Anfangszeit angezeigt. Danach wird die Zeit heruntergezählt, eine Zehntelsekunde nach der anderen. Dies erfolgt nach folgendem Algorithmus (Pseudocode)

solange

       nicht min und sek und zehntelsek gelich 0

    führe aus

        reduziere zehntelsek um 1

        falss dabei zehntelsek kleiner als 0 geworden ist

            setze zehntelsek auf 9 und

            reduziere sek um 1

            falls dabei sek kleiner als 0 geworden ist

                setze sek auf 59 und

                reduziere min um 1

Dabei wird immer dann die verbleibende Zeit angezeigt, wenn

·         sowohl die Anzahl der Sekunden durch 10 teilbar,

·         als auch die Anzahl der Zehntelsekunden 0,

·         als auch die Gesamtzeit noch nicht abgelaufen ist.

Unmittelbar nach Ablauf der Gesamtzeit werden die letzen beiden Anzeigen gemacht.

Die Ausgabe auf die Konsole erfolgt mit System.out.println(s), wobei s ein String Objekt ist.
