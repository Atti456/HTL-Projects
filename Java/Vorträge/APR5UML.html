Beziehungen/Relationen zwischen Klassen und UML (Unified Modeling Language)

Die UML bietet eine genormte Notation zur Beschreibung von objektorientierten Strukturen. Die UML ist ein von einem internationalen Gremium OMG (Obect Management Group) standardisiertes System, welches die besten Analyse und Entwurfsmethoden der Software Entwicklung zusammenfasst.
