Ein Erstes Beispiel

Einführung

Es wird zuerst eine Klasse Person entwickelt und übersetzt. Eine Klasse muss zur Benutzung in übersetzter Form vorliegen (Byte Code).

In einer zweiten Klasse PersonTest werden in einer bestimmten Methode, die der Start-Punkt des Programms ist, Objekte der Klasse Person instanziert und Nachrichten an diese Objekte gesendet. Auch diese Klasse wird kompiliert.

Dann wird das Programm gestartet.

Die Klasse Person

Die Struktur:

 

class Person   // Klassenkopf

//Klassenrumpf

{

            // Attribute

             private String vorname;

             private String nachname;

// Konstruktor

             public Person(String derVorname, String derNachname)

             {

                  vorname = derVorname;

                  nachname = derNachname;

             }

// Methoden

             public void aendereNachname(String neuerNachname)

             {

                  nachname = neuerNachname;

             }

             public String gibName()

             {

                  return vorname + " " + nachname;

             }

 }

 

Bevor die Klasse Person benutzt werden kann, muss sie compiliert werden. Wir gehen davon aus, dass sich der Quellcode in der Datei Person.java befindet. Man kann nun direkt in der Kommandozeile oder im JCreator kompilieren. Kompiliert man im Kommandozeilenmodus muß man die PATH Variabel erweitern und anschließend mit dem Befehl javac Person.java die Kompilierung einleiten. Im JCreator kann die Kompilierung im Menü Build mit Copile Project aufgerufen werden.

Die Klasse PersonTest

Die Klasse PersonTest hat die Aufgabe, einen kleinen Test auszuführen, in dem Objekte der Klasse Person erzeugt und Methoden für diese Objekte aufgerufen werden. In Java ist der Sartpunkt eines Programms immer eine bestimmte Methode mit vergegebenen Format. Sie hat den Namen main und ist eine Klassemethode. Klassenmethoden haben wie in C++ in Speichertyp static. Beinhaltet eine Klasse die Klassen-Methode main so ist sie eine sogenannte ausführbare Klasse.

Die Struktur:

 

class PersonTest

{

          public static void main(String[ ] args)

          {

                 Person p1 = new Person("Andrea", "Gruber");

                 System.out.println(p1.gibName());

                 Person p2 = new Person("Richard", "Frech");

                 System.out.println(p2.gibName());

              

                 p1.aendereNachname("Frech");

                 System.out.println(p1.gibName());

           }

}

 

Auch die Klasse PersonTest muss kompiliert werden, bevor sie verwendet wird. Man bringt die main() - Mehtode der Klasse PersonTest zur Ausführung, indem man in der Kommandozeile folgendes eingibt java PersonTest oder im JCreator im Menü Build den Punkt Execute Project auswählt.
